# 3dsketchmodul6insolidworks
Proyek ini mencakup penyelesaia , mulai dari Hydrolic Cylinder Half, Cylinder Jig, hingga Tooling Cylinder, dikerjakan dengan memperhatikan detail teknis, toleransi massa sebesar 1%, dan perubahan variabel desain (A, B, C, D) yang memerlukan fleksibilitas dalam pemodelan.
